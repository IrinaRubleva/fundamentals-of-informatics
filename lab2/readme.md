# Lab 2: Ряд Тейлора

## Задание
Вычислить значение функции $e^x \cos x$ в точке $x$ при помощи разложения в ряд:

$$e^x \cos x = \sum_{n=0}^{\infty} \frac{2^{\frac{n}{2}} \cos \frac{n\pi}{4}}{n!} x^n$$

где $|lax| < \infty$.

## Тип данных
`double` (спецификатор формата: `%lf`)




# Требования
### Программа №1
- **Вход:** $x$, количество членов ряда $n$
- **Выход:** значение функции

### Программа №2  
- **Вход:** $x$, точность $\varepsilon$
- **Выход:** значение функции, количество использованных членов ряда
